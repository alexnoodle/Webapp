Readme

Description

Our project is a simple site that can search through a dataset of all of the spacewalks up until 2013. You can search throughout the vehicle, country, and crew sections using the general search and you can search through each individual section using the advanced search page. The different parameters to search through are: Country, crew members, vehicle, date, and duration. You can also search through a range of dates or durations. The search function doesn't crash when it receives erroneous information and has buttons on all pages other than the main to bring you back to the main.

Features

General search: the query is compared against all country, crewmember, and vehicle entries and the appropriate results are displayed. Advanced search: You can search through individual sections, specifically crewmember, country, vehicle, duration, and date. It also allows for searching a range of dates or a range of durations. The results page is displayed as a table in the middle of the page that can be scrolled through without scrolling the background image.
